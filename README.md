# User Management System (MERN Stack)

A full-stack User Management System built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
This application allows users to be added, updated, deleted, searched, viewed in detail, and exported to CSV.  
It also supports profile image upload using Cloudinary.
